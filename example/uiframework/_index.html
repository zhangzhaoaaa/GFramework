Events

UI内部事件系统。

实例

BlaBlaBla


var gmp = new GMP({
    init: function() {
        this.on('eventName', function(args){
            console.log(args);
        });
    }
});
gmp.trigger( 'eventName', 1, 2, 3 );

on

on( name, fn )


    参数:

    name {String}事件名
    callback {Function}事件处理器。

    绑定事件。

off

off(name)


    参数:

    name {String}事件名
    解除事件绑定

trigger

trigger(name[, ...] )


    参数:

    evt {String}事件名
    {*}任意参数
    触发事件
